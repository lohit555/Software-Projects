# Software Projects

# 1) WorDex
A team-based C wordle game focused on modular program design, file handling, and testing. Includes core gameplay features such as word validation, scoring logic, and structured test cases.

# 2) GeoShield
A team-based hackathon project centered on a real-time disaster response web application. Focuses on collaboration, working with an existing full-stack codebase, and project integration.
